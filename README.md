# Chess by OpenCode

CLI chess game project organized with RUP Strikes Back.

## Getting Started

Requirements: Node.js 18+

Install:
```bash
npm install
```

Run CLI:
```bash
node src/cli.mjs
```

## Repository Structure
- src/: CLI and game logic (ESM)
- progress/: RUP artifacts
- RUPStrikesBack/: Submodule with process docs

## Development
- Conventional commits
- No secrets committed
- Keep examples copy-paste-able
